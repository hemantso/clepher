## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm

    ```sh
    npm install npm@latest -g
    ```

### Installation


1. Clone the repository

  ```bash
  git clone https://github.com/hemantso/clepher.git
  ```

2. Change directory

    ```sh
    cd clepher
    ```

3. Install NPM packages

    ```sh
    npm install
    ```

4. Start the local server

    ```sh
    npm start
    ```
